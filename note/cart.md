### 聊表操作

点击选择项选中一条，


点击全选按钮全部选中，

点击加减按钮可以改变商品数目

点击删除按钮可以删除商品
```
create table myfriend(

　　　　id int(11) not null auto_increment,

　　　　name varchar(20) not null,

　　　　email int(30) default null,

　　　　phone int(11) not null,

　　　　content varchar(100) not null

　　)ENGINE=InnoDB DEFAULT;
```
